# Workforce Analytics and Staff Distribution Dashboard for the Federal Ministry of Innovation, Science and Technology (FMIST)

## Overview

This project presents a Workforce Analytics and Staff Distribution Dashboard developed for the **Federal Ministry of Innovation, Science and Technology (FMIST)** using **Microsoft Excel, SQL, and Power BI**.

The project focuses on analyzing workforce data to provide insights into staff distribution, departmental allocation, grade level structure, and overall workforce composition. The dashboard supports management reporting, workforce planning, and data-driven decision-making.

## Tools Used

- Microsoft Excel
- SQL
- Power BI

## Key Features

- Staff Distribution by State
- Departmental Staff Analysis
- Grade Level Analysis
- Interactive Power BI Dashboard
- Workforce Summary Reports
- Data Cleaning and Validation
